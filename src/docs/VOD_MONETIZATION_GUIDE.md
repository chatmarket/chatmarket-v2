# VOD資産化システム（動画販売＆ライブアーカイブ）

## 概要
ChatMarketの**ライブ配信**を**永続的な収益資産**に変える統合システムです。
配信後、自動的にアーカイブが保存され、VOD（ビデオ・オン・デマンド）として販売できます。

---

## 🎯 4つの統合機能

### 1. **AWS IVS自動アーカイブ**
- **機能**: 配信終了 → HLSファイルが自動的にS3バケットに保存
- **実装**: `functions/enableIvsAutoArchive.ts`
- **メリット**: 手作業なし、100%自動化、インフラコスト最小化

**呼び出し例**:
```javascript
await base44.functions.invoke('enableIvsAutoArchive', { streamId: '123' });
```

---

### 2. **ペイウォール（購入ボタン）**
- **機能**: 有料動画には「購入して視聴」ボタンを配置
- **対応決済**:
  - **エールコイン** 💛 → 最速・手数料なし（推奨）
  - **クレジットカード** 💳 → Stripe経由（決済手数料3.6% + ¥40外出し）
- **実装**: `components/video/Preview30SecPaywallModal`

**仕様**:
- 無料動画は無制限再生
- 有料動画は30秒プレビュー → ペイウォール表示
- 購入後、フルアクセス可能
- 購入履歴はPurchaseエンティティに記録

---

### 3. **30秒プレビュー機能**
- **機能**: 冒頭30秒だけは誰でも無料視聴可能
- **UX**: 「この先は有料です」と購入意欲を高める
- **実装**: `pages/WatchVideo` + `components/video/Preview30SecPaywallModal`

**動作フロー**:
1. 動画プレイヤー開始 → 自動再生
2. 30秒経過時 → 再生停止 + ペイウォール表示
3. ユーザー購入 → ロック解除 + フル動画再生

---

### 4. **VOD管理ダッシュボード**
- **機能**: 配信者が自分の動画販売を一元管理
- **実装**: `pages/VodManagement`
- **表示内容**:

#### KPI Cards:
- 総売上（¥）
- 販売本数
- 平均価格
- 総視聴数

#### 概要タブ:
- 📊 **日別売上推移**: 過去30日の売上トレンド
- 🎯 **価格帯別分布**: 無料/¥100-500/¥501-1000 など

#### 動画一覧タブ:
| タイトル | 価格 | 視聴数 | 販売数 | 売上 | ステータス |
|---------|------|--------|--------|------|-----------|
| 動画A | ¥300 | 520 | 12 | ¥3,600 | ✓ 承認 |
| 動画B | 無料 | 1200 | - | ¥0 | ✓ 承認 |

#### 販売分析タブ:
- 📈 **動画別売上ランキング**: 収益貢献度TOP 10
- 💡 **インサイト**: 収益最大化のコツ

  "is_free": false,
  "view_count": 520,
  "moderation_status": "approved"

```json
{
  "title": "2024-04-27 配信",
  "auto_archive_enabled": true,           // ✅ 自動アーカイブ有効

  "archive_vod_enabled": true,            // アーカイブをVOD化
  "archive_vod_price": 500,               // アーカイブを¥500で販売
  "archive_vod_url": "s3://chatmarket-vod/stream-123/"
}
```

### Purchase エンティティ (既存)
```json
{
  "item_type": "video",
  "item_id": "video-456",
  "amount": 300,                 // 売上 ¥300
  "buyer_email": "user@example.com",
  "status": "completed",
  "payment_method": "coin"       // または "card"
}
```

## 💰 収益シミュレーション例

### シナリオ: 30分ライブ配信 + 価格¥500に設定

#### ライブ配信収益（その場）:
- 視聴者数: 50人
- 平均滞在時間: 20分
- エール: ¥5,000分 (10人 × ¥500)
- **ライブ収益**: ¥5,000 × 90% = **¥4,500**

#### アーカイブVOD化（継続）:
- 初月VOD売上: ¥500 × 20本 = ¥10,000
- **VOD収益 (月)**: ¥10,000 × 85% = **¥8,500**

#### 年間効果:
- ライブ（毎週1回）: ¥4,500 × 52週 = **¥234,000/年**
- VOD（累積）: ¥8,500 × 12月 = **¥102,000/年**
- **総年収**: **¥336,000** 🚀

> 💡 **ポイント**: 一度の配信で「その場のライブ収益」と「永続的なVOD資産」の二重取りが可能！

## ⚙️ 技術仕様

### AWS IVS 設定
- **地域**: `ap-northeast-1` (東京)
- **自動アーカイブ**: HLS形式 (`.m3u8` + `.ts`)
- **S3 保存先**: `s3://chatmarket-vod/{streamId}/`
- **有効期限**: 180日（ユーザー契約に応じて変更可）

### Stripe決済（ペイウォール）
- **手数料**: 3.6% + ¥40（ユーザー負担・透明表示）
- **対応決済**: カード / Apple Pay / Google Pay
- **処理時間**: リアルタイム

### エールコイン決済
- **手数料**: 0円（プラットホーム内決済）
- **実行速度**: 即座（推奨決済方法）

## 🎓 配信者向けのコツ

### 売上を最大化するには:

#### ✅ DO:
- **30秒プレビューで興味を引く**: 冒頭から結論を見せる
- **シリーズ化**: 続編が見たくなる構成にする
- **定期投稿**: 週1～2回のリズムで信頼感を構築
- **価格帯を分ける**: 
  - 新作: ¥500 (高品質)
  - 過去配信: ¥300 (手軽)
  - 番外編: ¥100 (気軽)
- **SNSで告知**: ツイッター・インスタで「新作VOD配信」をお知らせ

#### ❌ AVOID:
- ❌ 有料化直後に無料公開（購入者が激怒）
- ❌ 価格を毎週変更（購買心理が混乱）
- ❌ 低品質動画を高価格設定（評判が落ちる）

## 📌 よくある質問

### Q: ライブ中に「ながら配信」できるか?
**A**: はい。OBSモード（RTMPSで安定配信）とブラウザモード（スマホでも可）の両立が可能です。

### Q: アーカイブは何度でも再販売できるか?
**A**: はい。同じアーカイブを複数の価格タイプ（¥100・¥500・¥1000）で分割販売も可能です。

### Q: 視聴者が「購入後に削除」したら?
**A**: Purchaseレコードが「completed」のため、視聴権は消えません。refundの場合のみ視聴権が失効します。

### Q: 年間どのくらい売上が見込めるか?
**A**: 週1回の配信 + 平均¥300の価格で、年間**¥150,000～400,000**が目安（視聴者数による）。
