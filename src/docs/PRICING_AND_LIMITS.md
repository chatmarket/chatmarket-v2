# 価格設定・画質・時間制限ガイド【2026年4月更新版】

## 📌 全体方針

ChatMarketは、**最低価格15分15円**でコンテンツを販売し、運営収支を確保しながら、より多くのユーザーに楽しんでいただくプラットフォームです。

---

## 1️⃣ 価格設定ルール

### 動画販売・アーカイブ共通
- **最低価格：15分15円（税別）**
- **単位：15分刻み**
- **設定方法：配信者が自由に設定可能**

#### 価格例
| 長さ | 最低価格 | 推奨価格 |
|------|---------|---------|
| 15分 | ¥15 | ¥15～¥300 |
| 30分 | ¥30 | ¥30～¥600 |
| 60分 | ¥60 | ¥60～¥1,200 |

### 決済手数料（外出し）
- **Stripe決済：3.6% + ¥40（ユーザー負担）**
- **エールコイン決済：手数料なし（推奨）**

💡 **おすすめ：エールコイン決済を活用してください**
→ 手数料なし、即座に報酬が確定

---

## 2️⃣ アーカイブ配信画質

### 配信品質（固定）
- **解像度：720p (HD)**
- **フレームレート：24-30fps**
- **コーデック：H.264またはVP9**

### 理由
- AWS CloudFront転送コストを **15分あたり約0.3円** に抑制
- 15円販売でも確実に黒字を維持（5%=0.75円の利益確保）
- スムーズな再生体験を実現

### ユーザーメッセージ
> 「アーカイブ動画は、高画質を維持しつつスムーズな再生を実現するため、**720p (HD)** での配信となります」

---

## 3️⃣ 1日の視聴制限（時間制限ロジック）

### ルール
- **1ユーザーあたり1日最大60分の視聴・購入が可能**
- **リセット：毎日午前0時（UTC+9）**
- **対象：動画販売・アーカイブ販売の購入視聴**

### 実装方法
```javascript
// ローカルストレージで日次リセット管理
const DAILY_VIEW_LIMIT_MINUTES = 60;

// useDailyViewTimeLimit フックを使用
const { dailyViewedMinutes, remainingMinutes, isLimitReached, addViewTime } = useDailyViewTimeLimit(userId);

// 動画購入時に時間を加算
addViewTime(videoDurationMinutes);
```

### ユーザーメッセージ
> 「より多くの方に楽しんでいただくため、1日の合計視聴時間は **60分** を上限としております。明日またご利用ください 🎬」

## 4️⃣ 運営利益の計算（15円販売の例）

### 売上構成
| 項目 | 金額 | 備考 |
|------|------|------|
| ユーザー支払額（Stripe） | ¥55.40 | 15円 + 手数料(3.6% + 40円) |
| 配信者還元 | ¥15.00 | 販売価格100% |
| **運営利益** | **¥0.75** | 5%（転送コスト0.3円含） |

### 黒字の仕組み
1. **決済手数料を外出し** → ユーザーが負担
2. **アーカイブ画質を720pに固定** → AWS転送コスト削減
3. **1日60分制限** → サーバー負荷の平準化

### FAQ新規項目

#### Q. アーカイブ動画の画質について
**A.** アーカイブ動画は、スムーズな再生と運営コスト最適化のため、**720p (HD)** での配信となります。フル HD相当で視聴いただけます。

#### Q. 1日60分の視聴制限について
**A.** より多くの方に楽しんでいただくため、1日の合計視聴・購入時間は **60分** を上限としております。明日午前0時に自動的にリセットされます。

#### Q. エールコイン決済の方がお得ですか？
**A.** はい。Stripe決済では手数料（3.6% + 40円）がかかりますが、**エールコイン決済は手数料無料** です。また報酬が即座に確定されるメリットもあります。
