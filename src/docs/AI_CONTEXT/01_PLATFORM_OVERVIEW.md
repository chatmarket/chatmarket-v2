## 1. ビジョン

日本語向けのクリエイターエコノミープラットフォーム。  
配信者（ライバー・講師・占い師・アイドル等）が視聴者・ファンから直接収益を得られる場を提供する。

---

## 2. 対象ユーザー

**クリエイター側（配信者）**:
- ライブ配信者（雑談・ゲーム・音楽等）
- 占い師（チャット鑑定・ビデオ占い）
- オンライン講師（1対9クラス配信）
- アイドル・タレント（デジタルチェキ）
- コンサルタント・フィットネストレーナー等のエキスパート

**ユーザー側（視聴者・生徒）**:
- ライブ配信を視聴したい人
- 占い鑑定を受けたい人
- オンライン授業を受けたい人
- 推しにエール（投げ銭）を贈りたい人

---

## 3. 主要機能

| 機能 | 概要 |
|-----|------|
| ライブ配信 | AWS IVS を使用した1対多配信。有料（PPV）・無料対応 |
| VOD販売 | S3+CloudFrontでのオンデマンド動画販売（最低100コイン） |
| 1対1ビデオ通話 | WebRTC P2P。15分単位課金。延長機能あり |
| クラス配信 | Amazon Chime SDK。1対9（最大10名）。SchoolTicket認可 |
| チャット鑑定 | 占い師向け。2往復4通で自動クローズ。試し読みマスク機能 |
| エール送信 | 投げ銭（任意額）。クリエイターに85%還元 |
| ファンクラブ | Stripe Subscription 定期課金。ティア制 |
| グッズ販売 | 物理・デジタル商品販売。カスタムオーダー対応 |
| デジタルチェキ | アイドル向け。個別メッセージ入り写真販売 |
| イベントチケット | QR付き転売防止チケット |
| クラウドファンディング | Stripe Connect使用。プロジェクト単位支援 |
| コミュニティ | フォーラム・投稿・コメント |
| ダイレクトチャット | クリエイターへのDM |

## 8. 差別化要素

- **日本語特化**: UI・サポート・コンテンツモデレーション
- **クリエイター還元率**: Freeプランでも70%（競合比較で高水準）
- **プログレッシブ・インセンティブ**: 月間売上に応じて最大95%還元
- **エールコイン**: 1コイン=1円の透明性
- **多機能統合**: 配信・通話・鑑定・教育・グッズを1プラットフォームで提供
